# 📊 Análise Exploratória dos Estabelecimentos de Saúde de Santos (SP)

## 📖 Sobre o projeto

Este projeto apresenta uma **Análise Exploratória de Dados (EDA)** realizada com informações dos estabelecimentos de saúde do município de **Santos (SP)**.

O objetivo é aplicar técnicas de análise de dados utilizando **Python** e **Pandas** para explorar a base, identificar padrões, responder perguntas de negócio e extrair insights sobre a distribuição dos estabelecimentos de saúde no município.

Este projeto faz parte do meu portfólio de estudos em **Análise de Dados**.

---

## 🎯 Objetivos da análise

Durante a exploração da base de dados, foram respondidas as seguintes perguntas:

* Quantos estabelecimentos de saúde existem na base de dados?
* Quais são os 10 bairros com mais estabelecimentos de saúde?
* Quais CEPs possuem a maior quantidade de estabelecimentos de saúde?
* Quantos estabelecimentos possuem CNPJ próprio informado?
* Quantos estabelecimentos possuem CNPJ da mantenedora informado?
* Quantos bairros diferentes possuem pelo menos um estabelecimento de saúde?
* Quais logradouros concentram o maior número de estabelecimentos de saúde?
* Quais razões sociais possuem mais unidades cadastradas?
* Qual é o percentual de estabelecimentos concentrado nos 10 bairros com maior número de registros?
* Existem valores ausentes na base de dados?

## 🚀 Como executar o projeto

1. Clone este repositório:

   ```bash
   git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
   ```

2. Instale as dependências:

   ```bash
   pip install pandas jupyter
   ```

3. Abra o Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

4. Execute o notebook para acompanhar toda a análise.
